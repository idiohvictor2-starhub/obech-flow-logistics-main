# Obech Flow Logistics

A modern logistics and delivery management platform built with React and Vite.

---

# Project Overview

Obech Flow Logistics provides a complete logistics solution for:

- Bike Dispatch
- Van Delivery
- Truck Logistics
- Pickup Booking
- Shipment Tracking
- Customer Management
- Driver Management
- Vehicle Management
- Delivery Monitoring
- Admin Dashboard

---

# Technology Stack

- React
- Vite
- React Router
- React Query
- Tailwind CSS
- Local Storage (Development)
- Supabase (Production - Planned)

---

# Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
